# Celsius Holdings Financial Analysis

## Project Overview
Financial analysis of Celsius Holdings (NASDAQ: CELH) covering 2022-2024, including profitability turnaround assessment and competitive benchmarking against industry leader Monster Beverage.

## Key Findings
- Tracked transformation from $187M loss to $145M profit
- Identified 57% operating margin gap vs industry leader
- Analyzed 9x P/E premium despite operational inefficiencies
- ROE of 36.3% demonstrates efficient capital deployment

## Tools & Skills
- Excel: Financial modeling, ratio analysis, data visualization
- Financial Analysis: Margin analysis, valuation metrics, peer comparison
- Data Sources: SEC 10-K filings, Yahoo Finance
